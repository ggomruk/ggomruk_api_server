## Requirements

- MongoDB and RabbitMQ must be installed. Please refer to the official documentation for installation
- Refer to the .sample.env file to create a .env file

## Description

## Installation

```bash
npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
